📚 简便运算公式速查

📌 乘法运算律

  • 乘法交换律：a × b = b × a
  • 乘法结合律：(a × b) × c = a × (b × c)
  • 乘法分配律：a × (b + c) = a × b + a × c
  • 乘法分配律逆用：a × b + a × c = a × (b + c)

📌 加法运算律

  • 加法交换律：a + b = b + a
  • 加法结合律：(a + b) + c = a + (b + c)

📌 减法的性质

  • 减法性质1：a - b - c = a - (b + c)
  • 减法性质2：a - (b - c) = a - b + c

📌 除法的性质

  • 除法性质1：a ÷ b ÷ c = a ÷ (b × c)
  • 除法性质2：a ÷ (b ÷ c) = a ÷ b × c

📌 常用凑整方法

  • 25 × 4 = 100，看到25找4，看到4找25
  • 125 × 8 = 1000，看到125找8，看到8找125
  • 凑成整十整百：把数拆分成整十整百的形式

📌 常见简便运算技巧

  • 提取公因数：找出共同的因数，用分配律逆用
  • 拆分法：将一个数拆成两个数相加或相减
  • 补整法：将接近整十整百的数补成整数再计算
  • 带分数化假分数：便于约分和计算
